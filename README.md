# FUTURE_DS_01 – Business Sales Dashboard (Power BI)

## Internship Track
Data Science & Analytics – Future Interns

## Task Overview
This task focuses on building an interactive business sales dashboard using e-commerce data. The objective is to help business stakeholders understand sales performance, profit trends, and key drivers across different regions, categories, and time periods.

## Dataset
Superstore / E-commerce sales dataset (CSV format) containing:
- Order Date
- Sales
- Profit
- Quantity
- Category & Sub-Category
- Region
- Order ID

The dataset was loaded and cleaned using Power BI Power Query.

## Tools Used
- Power BI Desktop
- CSV dataset (E-commerce sales data)

## Work Done
- Cleaned and prepared raw sales data using Power Query
- Created KPI cards for:
  - Total Sales
  - Total Profit
  - Total Quantity Sold
  - Total Orders
- Built interactive visualizations:
  - Monthly sales trend
  - Sales by category
  - Profit by category
  - Sales by region
  - Sub-category performance
- Added slicers for:
  - Year
  - Region
  - Category
- Applied consistent formatting and color themes for better readability
- Enabled interactivity so that selecting filters updates all visuals

## Key Insights
- Sales performance varies significantly across regions and categories
- Certain product categories generate high sales but lower profit
- Seasonal trends are visible in monthly sales analysis
